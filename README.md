## Project Setup

### Install Dependencies
```bash
npm install
```

### Development
Start the development server:
```bash
npm run dev
```

### Build for Production
Build for different platforms:
```bash
# For Windows
npm run build:win
# For macOS
npm run build:mac
# For Linux
npm run build:linux
```
